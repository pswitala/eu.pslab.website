Zgodność NIS2

Przygotowujemy operacyjne audyty bezpieczeństwa IT by spełnić zasady dyrektywy unijnej. Dyrektywa NIS2 wymagania narzuca m.in. polityki backupów, szyfrowanie oraz logowanie alertów do CSIRT NASK.

Sieci LAN, WAN i WiFi

Rozwiązania Enterprise dostosowane do biur i hal: sieci Cisco konfiguracja (SD-WAN, Catalyst), jak również budżetowy standard z Ubiquiti UniFi instalacja sprzętu.

Fortinet Firewall Wdrożenie

Kombinacja Unified Threat Management (UTM) na bazie urządzeń FortiGate. Chronimy przed atakami zero-day, filtrujemy HTTPS, udostępniamy bezpieczny tunel VPN-SSL/IPsec.

Active Directory Wdrożenie

Kontrola tożsamości, GPO polityki haseł i LAPS. Active Directory wdrożenie wraz z administracją Windows Server — poprawne role Domain Controllera, DNS/DHCP.

Administracja serwerami Linux

Optymalizacja stosu LAMP, kontenerów Docker/Podman, skrypty bash i iptables oraz hardening systemu OS. Bieżąca konserwacja maszyn wirtualnych.

Audyty i Wdrożenia

Jak rozpocząć przygotowania do NIS2?

Proces zalecamy zacząć od gruntownego audytu bezpieczeństwa IT badającego bieżącą konfigurację sprzętową, fizyczną, procedury logowania, ciągłości działania oraz szkolenia pracowników. Zgodność NIS2 jest procesem, nie jednorazową konfiguracją firewall'a.
