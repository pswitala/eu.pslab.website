Power BI i hurtownia danych w Azure Synapse Analytics

Wyzwanie

Firma handlowa z 300+ pracownikami prowadziła raportowanie w dziesiątkach plików Excel rozproszonych po działach. Brak centralnej hurtowni danych oznaczał, że raporty sprzedażowe, finansowe i magazynowe były niespójne — każdy dział miał „swoją prawdę". Zarząd nie miał jednego widoku na kondycję firmy.

Rozwiązanie

Wdrożyliśmy pełny stos Business Intelligence na Azure:

  • Azure Synapse Analytics — centralna hurtownia danych integrująca ERP (Comarch), CRM, system magazynowy i dane z e-commerce.
  • Azure Data Factory — pipeline'y ETL do ekstrakcji, transformacji i ładowania danych z 8 źródeł (codziennie o 5:00).
  • Azure Data Lake Storage — raw data lake dla danych historycznych i plików źródłowych.
  • Power BI Pro — 35 interaktywnych dashboardów: sprzedaż (daily/weekly/monthly), marże, rotacja zapasów, cash flow, należności, KPI handlowców.
  • Row-Level Security (RLS) — każdy handlowiec widzi tylko swoje dane, manager widzi swój region, zarząd widzi wszystko.
  • Szkolenia użytkowników — 4 warsztaty self-service BI dla 50 osób.
