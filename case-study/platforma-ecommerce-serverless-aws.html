Platforma e-commerce serverless na AWS

? AWS?? 2022? 5 min czytania
Klient
Retailer online, sektor fashion
Branża
E-commerce / Retail
Okres realizacji
Q2–Q4 2022
Platforma
Amazon Web Services

Wyzwanie

Szybko rosnący sklep internetowy z branży fashion doświadczał regularnych awarii podczas szczytów ruchu (Black Friday, wyprzedaże sezonowe). Monolityczna aplikacja na dedykowanych serwerach nie skalowała się wystarczająco szybko, a koszty utrzymania stale rosły mimo niskiego ruchu poza sezonem.

Rozwiązanie

Zaprojektowaliśmy i wdrożyliśmy w pełni serverless architekturę na AWS, eliminując zarządzanie serwerami i włączając automatyczne skalowanie:

  • AWS Lambda — logika biznesowa w mikroserwisach (koszyk, płatności, zamówienia, inventory).
  • Amazon API Gateway — REST API z rate limiting i cachingiem.
  • Amazon DynamoDB — baza NoSQL z auto-scalingiem dla katalogu i zamówień.
  • Amazon CloudFront + S3 — CDN dla frontendu (Next.js SSG) i zasobów statycznych.
  • Amazon SQS + SNS — asynchroniczne przetwarzanie zamówień i notyfikacje.
  • AWS Step Functions — orkiestracja procesu realizacji zamówienia.

Efekt biznesowy: Dzięki eliminacji awarii w Black Friday 2022 klient zanotował 3x wyższe przychody niż rok wcześniej. Architektura pay-per-use pozwala płacić praktycznie zero w niskim sezonie.
