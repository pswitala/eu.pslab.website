Infrastruktura cloud na GCP dla firmy logistycznej

Wyzwanie

Firma logistyczna z flotą 500+ pojazdów potrzebowała platformy do śledzenia floty w czasie rzeczywistym, optymalizacji tras i analizy danych operacyjnych. Istniejący system oparty na starym oprogramowaniu desktopowym nie wspierał GPS tracking, a raporty generowano ręcznie w Excelu z opóźnieniem 24-48h.

Rozwiązanie

Wdrożyliśmy platformę na Google Cloud Platform zoptymalizowaną pod real-time data processing:

  • Google Kubernetes Engine (GKE) — mikroserwisy do przetwarzania danych GPS i orderów.
  • Cloud Pub/Sub — message broker do streamingu pozycji GPS z urządzeń IoT (1000+ wiadomości/s).
  • BigQuery — hurtownia danych z petabyte-scale analityką historyczną tras, kosztów paliwa i czasu dostawy.
  • Google Maps Platform — wizualizacja floty na mapie w czasie rzeczywistym i optymalizacja tras.
  • Cloud Run — microservices alertów i notyfikacji (opóźnienia, awarie, geofencing).
