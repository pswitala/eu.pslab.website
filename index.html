FAQ

Najczęściej zadawane pytania

Jakie usługi cloud computing oferuje PSLAB?

PSLAB oferuje kompleksowe usługi cloud computing na platformach AWS (Amazon Web Services), Google Cloud Platform (GCP), Microsoft Azure oraz OVH. Obejmują one projektowanie architektury chmurowej, migrację do chmury, zarządzanie infrastrukturą multi-cloud, optymalizację kosztów cloud, wdrażanie DevOps, CI/CD oraz Infrastructure as Code z Terraform.

Czy PSLAB wdraża rozwiązania sztucznej inteligencji i machine learning?

Tak, PSLAB specjalizuje się we wdrażaniu modeli machine learning, rozwiązań generative AI oraz autonomicznych systemów agentic AI. Nasz zespół posiada certyfikat AWS Machine Learning Specialty i doświadczenie w NLP, computer vision, deep learning oraz automatyzacji procesów biznesowych z wykorzystaniem AI.

Jakie certyfikaty posiada zespół PSLAB?

Nasz zespół posiada certyfikaty: AWS Solutions Architect Professional, AWS Security Specialty, AWS DevOps Professional Engineer, AWS Machine Learning Specialty, MCSA, MCSE, MCITP Administrator oraz MCITP Database Administrator. Łączymy tytuły naukowe (dr inż., mgr) z certyfikatami branżowymi.

Czy PSLAB oferuje obsługę informatyczną firm i bezpieczeństwo IT?

Tak, świadczymy kompleksową obsługę informatyczną firm obejmującą: sieci Cisco, Fortinet i Ubiquiti, bezpieczeństwo IT z SIEM i zgodność z dyrektywą NIS2, administrację serwerami Linux i Windows Server, bazy danych MS SQL 2025, PostgreSQL, MySQL i Oracle, Active Directory, ADFS, SSO, usługi RDP i obsługę drukarek sieciowych.

Jakie narzędzia Business Intelligence obsługuje PSLAB?

Specjalizujemy się w wiodących platformach BI: Microsoft Power BI, Tableau oraz Amazon QuickSight. Oferujemy projektowanie interaktywnych dashboardów, modelowanie danych, budowę pipeline'ów ETL, Data Warehousing oraz szkolenia z analizy danych i raportowania.

Gdzie znajduje się siedziba PSLAB i jaki jest obszar działania?

Siedziba PSLAB mieści się w Tychach (woj. śląskie) przy ul. Rolnej 22, 43-100. Obsługujemy klientów z całej Polski – Śląsk, Małopolska, Dolny Śląsk, Mazowsze i inne regiony. Realizujemy również projekty zdalnie.
